We are currently working on Lynx 2.8.8.

Lynx's version numbers have "dev", "pre" or "rel" suffixes. For example, "lynx2.8.8dev.1" is a development version as part of the successive versions culminating in "lynx2.8.8rel.1".

This directory contains archive copies of the most recent Lynx development source code. The links on this page (to tarballs or zip-files) match the layout of the "lynx2-8-8" breakout subdirectory.

Natural Language Support / Internationalization

Lynx supports multiple languages through use of gettext.
Some message catalogs from the translation project are bundled in the source distribution's po directory.
Also read the notes in About NLS.

Questions or comments can be directed to the lynx-dev@nongnu.org mailing list,
which is archived at http://lists.gnu.org/archive/html/lynx-dev/
